Skill Game Progress Bar(mastery criteria)

This is a progress bar component for a series of skill games. Each game consists of a given number of items, and in order to master the game, students need to meet a given number of correct rate after they finish all items. This design tries to solve the problem how to communicate the mastery criteria.

See the Pen progress bar 3 - show/hide by Lauren Wang (@laurenvast) on CodePen.

Character customization prototype

It is a part of a kids game I’ve worked on. This feature lets kids to pick a character and change the equipment of it. This customized creature will show up in different animated states in the main game. They can change the color of the character too, and some customization’s color will be dynamically changed based on the character's color.

I designed the customization categrary and layer rules, dynamic color strategy, and the state animations.

Tech used: After Effects/Bodymovin, HTML, CSS, JavaScript/lottie

(Try pick an animal type, then click HAPPY. And ask me to see more behind the scene magic sweat!)
